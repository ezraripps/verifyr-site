Verifyr is a npm package for verifying data, you can verify strings objects and arrays. It can define types, if a string or array includes something and much more, creator: arze website arze.io

Verifyr

Easily verify data in node.js


new Verifyr( "Verifyr is the best!" )
.is( "string" )
.has( "!" )
.valid()
